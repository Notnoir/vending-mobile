# Assets Images

## Required Files

### MediVendLogo.png

- **Purpose**: Main logo displayed on splash screen
- **Recommended Size**: 512x512 px or higher (square format)
- **Format**: PNG with transparent background
- **Design**: Healthcare/medical icon or logo
- **Location**: Place the logo file in this directory as `MediVendLogo.png`

If the logo file is missing, the app will fall back to a `health_and_safety` Material icon.
